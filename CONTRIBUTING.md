### Development

Install dependencies.

```
npm install
```

Run tests.

```
npm test
```

### Dev Dependencies

<li>
	<a href="https://github.com/mochajs/mocha" target="_blank">Mocha</a> - JavaScript test framework.
</li>
<li>
	<a href="https://github.com/chaijs/chai" target="_blank">Chai</a> - BDD/TDD assertion library.
</li>
